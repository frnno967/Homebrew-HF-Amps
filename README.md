# Homebrew-HF-Amps
Project to share designs for homebrew HF linear amplifier circuits by K5JCJ

Ham radio to me is all about building and modifying your own equipment. And given the availability of older commercial equipment, sometimes even it needs improvement to bring it up to safe and modern standards. 

This repository is where various design files will be posted. Some are designed with specific conversions in mind, but others are useful as general purpose support circuits for use in amplifiers. These designs are available for all to manufacture, use, or even sell. All I ask is for my attribution to note be removed. But feel free to ask questions or use these designs if they help your journey into QRO. 73 and see you on the bands!
